# HeroRPG Reference Collection — Source Index

Scraped from the Internet Archive Wayback Machine (web.archive.org) for herorpg.net,
a browser-based RPG active roughly 2005-2008. This index lists every file saved under
`reference/`, its original URL, the wayback timestamp used, an approximate word count
of the extracted text, and a status flag.

Status flags:
- **ok** — real game-related prose, usable as reference material
- **thin** — page loaded but contains little content (short stub, listing page, or near-empty)
- **404** — Wayback (or the origin server) returned Not Found for this URL/timestamp
- **login-only** — the page is a vBulletin/game "you are not logged in" wall; no gameplay content behind it was retrievable from a public archive

  (empty at every available snapshot). Everything else yielded genuine period content or an

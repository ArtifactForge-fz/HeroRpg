Main Page - Project: Genesis Manual Main Page From Project: Genesis Manual Jump to: navigation , search Welcome to the HeroRPG Gameplay Manual Here you will find resources and information for playing the game and learning how to play the game. This information has been graciously authored by our players and staff. Please do not deface the manual if you choose to edit the pages. Please do not provide false information. Last, but not least, do not write articles that allow players to cheat in any shape, sort, or form. This includes but is not limited to: Item lists; monster lists; other content lists; exploits; duplication methods; and scams. You may use the information here as you see fit. If you have any contributions to make, you may add or edit pages at will, so long as the information added/edited is constructive, positive, and truthful. If there is a page you would like added that does not exist, you may create it by first searching for the page in the search box to the left, then clicking the "Create this page" link. Changes and Updates Recent changes to the game can be found in Recent Updates . Upcoming changes can be found in Upcoming Updates . Reporting Bugs If you have a bug to report, please check the Known Issues page first. If your bug is not listed, you may add it to the list of Current Bugs.

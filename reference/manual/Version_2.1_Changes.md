Version 2.1 Changes - HeroRPG Manual Version 2.1 Changes From HeroRPG Manual Jump to: navigation , search - Addition of Alchemy - Addition of Classes - Added 15 new techniques for the first 3 classes - Addition of the Spirit Shrine - Addition of Buffs - Added over 20 buffs to the Spirit Shrine for purchase - Added an "Anima Shard" currency for use with buffs - Addition of Arkan Race - 30 levels of content for the Arkan Race - 10 levels of new content for players level 30+ - 5 new quests for Human and Arkan races (30+) - Added synthesis shop items for levels 30+ - Added energy stones to level 30+ shop - Added new predefined transmutation recipes - New quests for Arkan (1-30) - Monsters can be attacked without viewing their dialog box - New theme "Evo" - Added tabs to the techs page to sort techniques by type - Added the ability for monsters to use techniques - Added 24 new techniques unique to monsters - Over 45 monsters now use techniques - Rebalanced level 20-30 monsters to accurately reflect dev. equations - Added new detrimental effects (Poison, Haunting, Curse) - Removed old status system (Blind, Silence) - Increased the difficulty of many boss fights - All items show long stat names instead of abbreviations - Tech dialog boxes show "effective damage" which figures in intelligence - Item dialog boxes show item components (for level 30+ items, on the "Misc" tab) - Shops and synthesis shops show a welcome message - You can no longer go over your weight limit through the vault - Academy categories break onto new lines so that only 3 are shown per row -
